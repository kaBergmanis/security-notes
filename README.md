Rīki tikai uzkrājas gadu gaitā. Reizēm viņus vajag lietot, reizēm padalīties, tāpēc salikšu strukturēti un publiski pieejami.

## Avoti
Viss, kur informācija ienāk regulāri - ziņu portāli un twitter konti, teiksim.

## Regulējumi
Likumi un likumu dažādi atbalstošie akti un faili.

## Rīki
Tā kā lietas, kur mēs varam kaut ko iebakstīt un dabūt kaut ko atpakaļ. Pārbaudīti strādājoši.
